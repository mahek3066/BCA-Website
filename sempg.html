1. Computer Basics and PC Software

Credits: 4

Objective: Master fundamentals of computers and essential software tools.

  • Computer System: Hardware components, software types, operating systems.
  • MS Office Suite: Word, Excel, PowerPoint for productivity.
  • Practicals: Hands-on exercises with 60% weight in assessment.

2. Mathematics

Credits: 4

Objective: Build a strong mathematical foundation for computer science.

  • Algebra: Sets, relations, matrices.
  • Calculus: Basic limits, derivatives, integrals.
  • Discrete Mathematics: Logic, graphs, probability.
  • Practicals: Problem-solving with 100% theory focus.

3. Programming in C/C++

Credits: 4

Objective: Learn procedural programming languages.

  • Basics: Variables, data types, control structures.
  • Advanced: Functions, arrays, pointers, file handling.
  • Practicals: Coding labs and debugging exercises.

4. English/Communication Skills

Credits: 4

Objective: Enhance language and communication abilities.

  • Grammar: Parts of speech, tenses, vocabulary.
  • Writing: Essays, reports, emails.
  • Speaking: Presentations, discussions.
  • Practicals: Oral and written assignments.

5. Web Designing/Web Technology

Credits: 4

Objective: Introduction to web development basics.

  • HTML: Structure, tags, forms, tables.
  • CSS: Styling, layouts, responsive design.
  • Principles: UI/UX, tools, SEO basics.
  • Practicals: Build static websites and projects.
